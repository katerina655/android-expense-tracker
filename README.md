## Android Expense Tracker

This is a simple Android application for tracking daily, weekly, and monthly expenses.  
The user can record the amount of money spent each day, and the application provides statistical summaries based on the stored data.

The application uses **SharedPreferences** for data storage, without any external databases.

## Features

- Add daily expenses.
- View total expenses for each week and month.
- View average daily and weekly expenses.
- View the day or week with the highest and lowest expenses.
- Simple and user-friendly interface.

---

## Technologies Used

- **Java**
- **Android Studio**
- **SharedPreferences**
- **XML Layouts**

## How Data is Stored

The application stores all data locally using the **SharedPreferences** API.  
This allows quick access without requiring a database or internet connection


## How to Run the Project

1. Download or clone the repository.
2. Open the project in **Android Studio**.
3. Allow Gradle to sync.
4. Run the application on a real device or emulator.
